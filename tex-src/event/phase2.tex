\subsection*{ステップ・目的}
\begin{flushleft}
\begin{enumerate}
  \item weekday 係数を自己適応的 EWMA\,(\(\lambda_{\text{wd}}\)) で平滑。
  \item 決算日の近傍 \(\pm2\) 日まで減衰する係数 \(w_d\) を適用。
  \item 市場係数は日経平均VIを加え、42 日相関で EWMA 更新。
\end{enumerate}
\end{flushleft}
